### Tic Tac Toe ###
By Jacob Moyle, 2015

---

An unbeatable Tic Tac Toe computer.

#### Instructions:
* Clone this repo:
  * Run `git clone https://github.com/jacobmoyle/TicTacToe`
* From the app directory:
  * Run `ruby runner.rb`

#### About:
* This game has three game modes:
  * Computer v. Computer
  * Player v. Player
  * Player v. Computer
